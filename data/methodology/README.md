# Methodology

Versioned scoring grid (CC BY-SA 4.0): 24 indicators, 5 pillars, weights, normalization thresholds with their references, confidence rules.

- One file per version (`v0.1.0.json`…), git-tagged at publication — the tag is the timestamped proof of anteriority.
- Every change = version bump + signed CHANGELOG entry with a rationale. No silent weight edits, ever.
- Provisional values are marked `PROVISIONAL` and are purged before the v0.1.0 freeze.
